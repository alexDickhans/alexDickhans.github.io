Raid The Romans

Raid the Romans is a project meant to illustrate ancient Roman military strategy, it is a videogame made with Python Processing. This is our Social Studies project for our unit on Ancient Rome

You play the game by clicking soilders to kill them and buying archer towers to boost up your health

The game is infinite so there is no winning but youhave braggring rights if you are highest on the leaderboard

 Leaderboard Score Submission Form

Update 4-5-20 Currently we are working on moving this from the less stable Python Processing framework to the more stable Processing Java framework to make the game less buggy.
